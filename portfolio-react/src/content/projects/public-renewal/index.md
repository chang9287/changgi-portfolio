# Overview

건설기술교육원의 기존 공공기관 홈페이지는 다양한 정보가 한 페이지에 혼재되어 있어 사용자가 원하는 정보를 빠르게 찾기 어려운 구조였습니다. 특히 정보가 명확한 기준 없이 배치되어 있어 탐색 과정에서 불필요한 시간과 노력이 발생하는 문제가 있었습니다.

그래서 이러한 불편함을 해소하기 위해 정보 구조를 사용자 기준으로 재정리하고, 주요 콘텐츠의 우선순위를 재배치하여 보다 직관적인 탐색이 가능한 UI로 개선하는 것을 목표로 프로젝트를 진행했습니다.


# Problem & Solution

## Problem

교육원 사이트의 핵심인 교육 신청은 메인 페이지 내에 부재했으며, 주요 공지사항 또한 하단에 소외되어 있었습니다. 실제로 이용할 땐 원하는 정보를 찾기 위해 불필요한 탐색을 반복해야 했습니다.

이러한 정보 배치의 불균형을 해결하기 위해 개선이 필요하다고 생각했습니다.

## Solution

방문자의 주요 목적인 교육 신청과 공지 확인이 원활하도록 메인 비주얼 하단에 핵심 메뉴 4개를 아이콘으로 배치하고, 이어지는 영역에는 수강신청 섹션과 공지사항을 순차적으로 구성했습니다.

이를 통해 메인에서 소외되었던 핵심 정보들의 시각적 위계를 사용자 중심으로 재정립하는 방향으로 설계했습니다.

## Result

이러한 설계의 변화로 사용자가 정보를 찾기 위해 겪어야 했던 불필요한 탐색을 줄임으로써 동선을 획기적으로 단축시키게 되었습니다.

또한 정보 배치의 불균형을 해소하여 처음 방문하는 사람들도 원하는 정보에 즉시 도달할 수 있는 직관적인 환경을 구축하게 되었습니다.

나아가 반응형 UI를 통해 모든 디바이스에서 설계 의도가 일관되게 유지되도록 구현하였으며, 결과적으로 서비스의 접근성과 완성도가 모두 높아지게 되었습니다.


# Main Feature

- Swiper 라이브러리를 활용한 자동 재생 슬라이드 구현
- JavaScript로 페이지네이션 UI 구현
- 클릭 이벤트 기반 아코디언 인터랙션 구현
- 탭 및 중첩 탭 구조(Tab in Tab) 설계 및 구현
- 반응형 레이아웃 적용


# What I Learned

단순히 보기에 좋은 디자인을 만드는 것을 넘어, 사용자의 이용 목적에 따른 정보 우선순위 배치가 UI/UX의 핵심 중 하나라는 것을 깊이 체감하게 되었습니다.

기획 단계에서 발견한 문제점을 논리적인 설계로 해결하는 과정을 통해, 개발자로서 사용자 중심의 사고가 코드의 방향성을 결정한다는 점을 깨닫게 되었습니다.
